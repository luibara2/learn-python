# Learn Python (farm coding game)

**Status: work in progress — not finished yet.** Features and behavior will change.

This is a small **Godot 4** game in the spirit of [*The Farmer Was Replaced*](https://store.steampowered.com/app/2060160/The_Farmer_Was_Replaced/): you write code to control a **drone on a farm grid** — till soil, plant crops, harvest, move around, and manage a simple inventory. The scripting language is **Python-like** and runs through a custom tokenizer, parser, and interpreter in this repo (not CPython embedded in the game).

Good fit if you want to practice **read–write–run** loops with farm automation puzzles, similar to the commercial game’s vibe, in an open project you can hack on.

## Running the project

1. Install [Godot 4.x](https://godotengine.org/download) (project targets 4.6 in `project.godot`).
2. Open this folder as a project in the Godot editor.
3. Run the main scene (`app.tscn`).
